nvme-flush(1)
=============

NAME
----
nvme-flush - Flush command.

SYNOPSIS
--------
[verse]
'nvme flush' <device> [--namespace-id=<nsid> | -n <nsid>] [--force | -f ]

DESCRIPTION
-----------
The Flush command shall commit data and metadata associated with the
specified namespace(s) to nonvolatile media. The flush applies to
all commands completed prior to the submission of the Flush command.
The controller may also flush additional data and/or metadata from
any namespace.

OPTIONS
-------
-n <nsid>::
--namespace-id=<nsid>::
	Specify the desired namespace id for this command. Defaults to 0x0.
	if VWC supports NSID 0xffffffff, and this field given is  0xffffffff,
	indicating flush for all namespaces.

-f::
--force::
	Force flush all the namespaces irrespective of the VWC support for
	NSID 0xffffffff.
